Siglas e TERMOS TECNICOS

ARPU = receita media por usuário
(quantas vendas o cliente compra x valor da compra)

GROSS MARGIN = LUCRO BRUTO

LTV = ARPU X GROSS MARGIN

CAC = Custo de aquisição do cliente

ROI= LTV / CAC (retorno sobre o investimento)

viéis cognitivos = formas de marcar as pessoas

CPM= custo por mil visualização

CPC= custo por clique

Aquisição = consquistar a pessoa

Monetização = convencer a pessoa a gastar

FAST TRAFFIC = trafico rapido para testar o produto e resultados rapidos

DEEP TRAFFIC = trafico profundo para campanhas mais calculas e resultado ao longo prazo

CRO - Otimização da taxa de conversão

RFM = RECENCIA, FREQUENCIA, VALOR MONETÁRIO

BI = Business Intelligence 

Linguagem DAX = Linguagem do Power Query

Linguagem M = Linguagem do Power Query

Power Query = poder automatizar o Power BI e Excel, importar conteudo do SQL

Power Query não tem CTRL + Z,tem "Voltar na linha do tempo"

Tabela Fato = Algo que aconteceu, exemplo: uma tabela de vendas
	na tabela fato deve ter somente o necessário	

Tabela Dimensão = unico valor linha, valores unicos por linha, exemplo: registro de clientes
	na tabela dimensão é onde fica os dados complementares	

SDR = menos tecnico com o objetivo de analizar o cliente se é bom ou não

BDR = Mais tecnico, Busca pessoas que nunca se cadastraram e tenta converter vendas

LDR = Menos Tecnico, Filtra os cliente para o BDR 

Closer = mais tecnico, só conversas com clientes bons que vem do SDR, marcas bastante reuniões

Farm/CS =  Objetivo permanecer com clientes para gerar compras constantes

Field Sales = Conversa diretamente com os diretores das maiores empresas

SQL = Sales qualified Lead (lead de vendas qualificadas)

ICP = ideal customer profile (cliente do perfil ideal)

Gatekeeper = funcionario do cliente que verifica o pq você está mandando mensagem

CRM = "Customer Relationship Management" (Gestão de Relacionamento com o Cliente)

FOLLOW-UP = acompanhamento de um processo após a execução da etapa inicial 

SKU = Stock Keeping Unit (Unidade de Manutenção de Estoque) unidade de produtos

GROWTH = CRESCIMENTO, CRESCER EMPRESA
